# 🚗 Automotive Market Insights: Used Car Data Analysis

## 📌 Project Overview
This project focuses on performing comprehensive Exploratory Data Analysis (EDA) on a real-world dataset of used car listings (`car_prices.csv`). The primary objective is to extract meaningful business intelligence regarding market trends, vehicle depreciation, and demand-supply dynamics in the used automobile sector. 

## 🎯 Objectives
* **Data Ingestion & Quality Profiling:** Load, inspect, and clean the dataset by handling missing values, anomalies, and duplicate records.
* **Data Wrangling:** Use Python's Pandas library to filter, group, and summarize large sets of automotive data.
* **Exploratory Data Analysis (EDA):** Extract actionable insights through targeted queries (e.g., pricing by brand, condition impact, value-for-money metrics).
* **Data Visualization:** Build intuitive charts to visualize feature correlations, geographic sales distributions, and historical pricing trends.

## 🛠️ Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn

## 📊 Key Features & Insights Extracted
* **Price vs. Condition:** Analyzed how vehicle condition scores directly impact the final selling price.
* **Depreciation Trends:** Visualized the inverse relationship between odometer readings/vehicle age and market value.
* **Geographic Hotspots:** Identified the top states for used car sales and mapped regional pricing differences.
* **Brand Valuation:** Highlighted the top "value-for-money" makes for vehicles in excellent condition and calculated average selling prices across different manufacturers.
